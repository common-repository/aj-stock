AJ 주식 플러그인은 한국 거래소에 등록된 모든 기업들의 주가 정보를 표현하며, 모든 항목들은 실시간으로 업데이트 됩니다.
화면 새로 고침 없이 자동으로 설정된 업데이트 주기에 따라 실시간으로 업데이트 됩니다.

== Description ==
== 주식 차트(일봉/1분) - Pro Version ==
* 기본으로 일봉 캔들 차트와 1분단위 라인차트가 제공됩니다.
* 설정을 통하여 거래량 차트와 차트 범위등을 상세히 설정할 수 있습니다.
* 1분마다 자동 업데이트 됩니다.
* 화면의 깜박임 없이 변경된 사항이 자동으로 반영됩니다.

== 실시간 주가 정보 ==
* 1분 단위로 자동 업데이트 됩니다. 화면의 깜박임 없이 변경된 사항이 자동으로 반영됩니다.

== 주가 상세 정보 ==
* 주가에 대한 자세한 정보를 제공합니다.
* 전일가, 상한&하한가, 액면가, 52주 최고/최저가, PER 값을 표시해줍니다.

== 거래원 정보 ==
* 매도/매수 상위 5개의 거래원에 대한 거래원과 거래량을 실시간으로 표현해줍니다.

== 위젯 ==
* 심플하고 깔끔한 위젯도 표현됩니다.

== Installation ==
1. aj-stock.zip을 다운로드 받아 압축을 풀고 **/wp-content/plugins/** 에 업로드 합니다.
2. 플러그인 메뉴에서 AJ Stock의 플러그인을 활성화 합니다.
